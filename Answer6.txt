git log показывает список всех выполненных коммитов, отсортированных по дате добавления.
git show используется для просмотра информации о коммите в репозитории Git. Она показывает информацию о последнем коммите, включая его хэш (SHA-1), автора, дату коммита и сообщение коммита. Кроме того, она также показывает изменения, внесенные в файлы в рамках данного коммита.
git blame выводит файл и рядом с каждой строчкой показывает того, кто ее менял и в каком коммите.
git grep ищет совпадение с указанной строкой во всех файлах проекта. Это очень удобная команда для быстрого анализа из терминала. Она удобнее обычного grep, потому что знает про игнорирование и не смотрит в директорию .git, а еще умеет искать по истории
